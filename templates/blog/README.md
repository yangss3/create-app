# A blog template powered by VitePress and Vue 3

## Features

- [x] Vue 3
- [x] Vite 2
- [x] VitePress
- [x] markdown
- [x] TypeScript
- [x] `vite-plugin-windicss` for WindiCSS
- [x] `unplugin-vue-components` for component auto importing
- [x] `unplugin-icons` for icons
- [x] Dark mode
- [x] ESLint
- [x] Conventional commits
- [x] SSG
